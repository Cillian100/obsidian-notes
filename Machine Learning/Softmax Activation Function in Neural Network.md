## Properties of the Softmax Function 
- **Output Range:** the softmax function guarantees that the output values lie between 0 and 1, satisfying the definition of the probabilities 
- **Sum of Probabilities:** the sum of all outputs from the softmax function always equal 0
- **Interpretability:** softmax transforms the raw outputs into probabilities
